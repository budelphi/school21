--> Что должна делать функция?

Функция должна считывать новую строку из файла (из fd в line). 

--> Что должна возвращать функция? 

1: Считана строка
0 : EOF достигнута
-1: Произошла ошибка

* Например, это может быть полезно когда нужно прочитать весь документ построчно
или создать цикл основанный на возвращаемом значении 

--> Как работает функция?

Функция принимает file descriptor для чтения из файла, записывает новую строку в line
и возвращет какое-то значение.

Размер буфера задается во время комплиции.

--> В чем трудности?

Буфер может быть меньше считываемой строки - тогда строку нужно будет склеивать.
Буфер может быть больше считываемой строки - тогда часть считанных данных нужно будет отсечь.
Каретка может остаться не в начале новой считываемой строки. 

--> Как работает моя программа?

Все основные функции происходят в get_line, в то время как get_next_line является просто оберткой,
а остальные функции носят вспомогательную роль.
